Especificacao 1.3 - Bem vindo

Visando melhorar a usuabilidade do sistema, deve-se exibir uma mensagem de boas vindas após o usuário mesmo efetuar o login.

Exemplos:
Dado que: Sou um usuario chamado João do sexo MASCULINO
Quando: Ao logar no sistema
Entao: A mensagem João, seja bem-vindo! deve ser exibida.
Dado que: Sou um usuario chamado Maria do sexo FEMININO
Quando: Ao logar no sistema
Entao: A mensagem Maria, seja bem-vinda! deve ser exibida.
