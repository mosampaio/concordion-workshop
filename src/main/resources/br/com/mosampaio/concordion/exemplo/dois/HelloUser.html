Especificacao 1.2 - Hello User

Visando melhorar a usuabilidade do sistema, deve-se exibir uma mensagem de boas vindas após o usuário mesmo efetuar o login.

Exemplos:
Given (Dado que): Sou um usuario chamado Marcos
When (Quando): Ao logar no sistema
Then (Entao): A mensagem Hello Marcos deve ser exibida.
